# XSD do eSocial S-1000 (S-1.3)

Coloque aqui o arquivo oficial do esquema XSD do evento S-1000 correspondente à versão S-1.3 (nome comum: `v_S_01_03_00.xsd`).

- Caminho esperado pelo validador: `schemas/esocial/s1000/v_S_01_03_00.xsd`
- Instale a dependência para validação: `npm install xsd-schema-validator`

Após adicionar o XSD e a dependência, o validador executará a checagem de esquema e reportará inconsistências como erros.

Observação: O arquivo XSD é distribuído pelos canais oficiais do eSocial. Não incluímos o arquivo neste repositório por questões de tamanho/atualização/licenciamento.
